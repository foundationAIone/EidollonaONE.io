# Time-Lens (line · loop · now)

- **line**: trend/evolution across days (KPIs and deltas).
- **loop**: daily/12-phase cycle view (restart-safe, cyclic runbooks).
- **now**: present readiness (SE4.3: readiness/impetus/risk/RA/Wings).

Use `/api/hud/signals?time_lens={now|line|loop}`. Epoch markers define new "beginnings" for analytics.
